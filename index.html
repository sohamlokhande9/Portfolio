About Me

Hey, I'm Soham

I'm a first-year Electronics and Telecommunication Engineering student at MIT Academy of Engineering, Alandi, Pune. Honestly, I got into ENTC because I've always been fascinated by how electronic devices actually work under the hood — and engineering felt like the right path to figure that out.

These days I spend a lot of time tinkering with Arduino projects, writing code that does something physical in the real world (motors spinning, sensors detecting, stuff actually moving). There's something really satisfying about that.

I'm also getting into web development — HTML, CSS, and slowly working my way into programming languages like C and C++. I'm at that stage where everything is new and exciting, and I'm trying to absorb as much as I can.

Things I've Built

My Projects

First-year projects, but I'm proud of every one of them.

🤖
01

Automatic Obstacle Avoiding Robot

Built a robot that can navigate around obstacles on its own — no remote, no manual steering. It uses an HC-SR04 ultrasonic sensor to detect what's in front of it and makes decisions using an Arduino Uno to control two DC motors. Took a lot of trial and error with the motor logic, but seeing it actually dodge things was genuinely exciting.

Arduino Uno HC-SR04 L298N Driver DC Motors C++
💡
02

Smart LED Controller

An Arduino Uno based project where LEDs respond to sensor input — brightness changes based on ambient light using a LDR sensor. It sounds simple, but this was one of my first projects where I understood how analog sensors actually communicate with microcontrollers. A good starting point that taught me more than I expected.

Arduino Uno LDR Sensor PWM Control C++
🌐
03

Personal Portfolio Website

This very website — my first real web project. Built entirely with HTML, CSS, and JavaScript, no frameworks. I wanted to make something that actually reflects my personality rather than just copy a template. Learned a ton about layout, animations, and how browsers actually render pages.
